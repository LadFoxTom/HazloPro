# Translation Keys Overzicht

Dit document geeft een overzicht van alle translation keys voor de Spaanse (es) en Engelse (en) vertalingen.

## Structuur

Alle translation keys zijn genest in objecten. De structuur is:
```
translations.es.{key} of translations.es.{category}.{key}
```
